
DataSyncTools
============
-------
>Java Swing  
>some icons

功能特性
---------
图形界面的一个小工具，依赖于Java环境，目前仅支持MySQL数据库间进行数据同步。
* 支持数据全量同步
* 支持数据和表结构全量同步
* 支持保存上一次进行同步的数据表，解决其他工具每次都要选表的麻烦。
* 支持检测数据表是否存在

环境依赖
---------
>Java 8+

目录结构描述
-------------
>dist: 目录下有打包好的可运行jar文件
>config：配置
>>config.xml：UI设置参数的持久化  
>>zh-cn.properties：语言，国际化
